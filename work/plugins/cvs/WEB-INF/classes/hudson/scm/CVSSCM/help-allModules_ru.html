Имя (имена) модуля(ей) в заданной CVSROOT для checkout. Несколько модулей могут быть указаны с использованием пробела в качестве разделителя. Если имя вашего модуля содержит пробел, вы можете заэскейпить его с помощью обратного слэша: '\ '. Например, "foo\ bar zot" указывает на два модуля: "foo bar" и "zot".
