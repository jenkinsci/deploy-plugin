Если установлено, то Hudson будет запускать CVS с ключом -r для сборки указанной ветви. Иначе будет осуществляться сборка из среза основной ветви кода.

Обычно Hudson запускает cvs checkout и update с ключом -D date, чтобы быть уверенным в получении когерентного среза репозитория и лучшей воспроизводимости, но это поведение вызовет конфликт, если имя ветви, указанное здесь, на самом деле является именем метки в репозитории (tag). Указав Hudson, что это имя является именем метки, вы избавитесь от этой проблемы.
