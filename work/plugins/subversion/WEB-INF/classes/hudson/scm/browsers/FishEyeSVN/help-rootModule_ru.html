Укажите корневой модуль Subversion, который использует FishEye. Например для http://fisheye6.cenqua.com/browse/ant/, это поле должно содержать ant, так как он отображает содержимое директории "/ant" репозитория ASF. Если FishEye сконфигурирован для отображения всего репозитория, оставьте это поле пустым.
