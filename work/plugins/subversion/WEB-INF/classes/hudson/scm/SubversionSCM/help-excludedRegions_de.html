Ist diese Option angewählt und das Projekt so konfiguriert, dass Hudson regelmäßig ein SVN-Repository nach Änderungen befragt, so werden dabei die hier angegebenen Dateien und/oder Verzeichnisse bei der Bestimmung, ob ein neuer Build ausgelöst werden soll, ignoriert.

Jeder Ausschluss kann reguläre Ausdrücke verwenden und muss ein einer eigenen Zeile stehen.

Im folgenden Beispiel wird kein neuer Build ausgelöst, wenn nur html/jpeg/gif-Dateien geändert wurden. 

   /trunk/myapp/src/main/web/.*\.html
   /trunk/myapp/src/main/web/.*\.jpeg
   /trunk/myapp/src/main/web/.*\.gif

Mehr zu regulären Ausdrücken....
